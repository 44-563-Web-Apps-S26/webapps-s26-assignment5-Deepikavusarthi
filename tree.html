Responsive Behavior Explanation

The tree names row breaks first when the browser window is reduced. This happens because the tree names use col-md, which keeps the columns in one row only on medium screens and larger. When the screen becomes smaller than the medium breakpoint, these columns stack vertically.

The letters row uses col-sm-2, which keeps the layout horizontal on small screens and larger. Therefore, the letters row stays horizontal longer and breaks after the tree names row.
